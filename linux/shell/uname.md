# uname

-a 显示所有信息
-i 显示硬件平台
-m 显示硬件架构
-n 显示主机名称
-v 显示内核版本
-r 显示内核发行版本号
-s 显示内核名称

```bash
uname -a
Linux edd870dcacc8 5.15.49-linuxkit-pr #1 SMP PREEMPT Thu May 25 07:27:39 UTC 2023 aarch64 aarch64 aarch64 GNU/Linux

uname -i
aarch64
uname -r
5.15.49-linuxkit-pr
uname -s
Linux
uname -m
aarch64
uname -v
#1 SMP PREEMPT Thu May 25 07:27:39 UTC 2023
```
